## 四、v5.0 新增功能：人工接管

### 4.1 功能说明

v5.0 新增人工接管功能，允许管理员在必要时介入AI对话：

- **访客端**：聊天窗口底部新增"🙋 转人工客服"按钮
- **管理后台**：新增"在线对话"页面，实时显示访客列表
- **接管流程**：管理员点击"✋ 接管对话" → AI暂停 → 人工回复
- **释放流程**：管理员点击"🔓 释放对话" → 恢复AI自动回复
- **状态标记**：访客列表显示 AI模式（蓝色）/ 人工接管（绿色）/ 请求人工（红色）

### 4.2 数据库表（human_replies）

```sql
CREATE TABLE human_replies (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  visitor_id text NOT NULL,
  content text NOT NULL,
  is_read boolean DEFAULT false,
  created_at timestamptz DEFAULT now()
);
```

### 4.3 系统指令

| 指令 | 含义 |
|------|------|
| `[SYSTEM:ADMIN_TAKEOVER]` | 管理员接管对话 |
| `[SYSTEM:ADMIN_RELEASE]` | 管理员释放对话 |
| `[SYSTEM:USER_REQUEST_HUMAN]` | 用户请求人工 |
| `[客户消息] xxx` | 人工模式下的客户消息 |
