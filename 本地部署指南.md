# AI 智能客服系统 - 本地部署指南

## 方式一：本地电脑运行（开发测试，免费）

### 第一步：准备环境

1. **安装 Node.js**（版本 18+）
   - 下载地址：https://nodejs.org/ （选 LTS 版本）
   - 安装完成后，打开终端运行 `node -v` 确认

2. **获取 Kimi API Key**（免费 15 元额度）
   - 访问 https://platform.moonshot.cn
   - 用手机号注册账号
   - 进入「API Key 管理」→ 点击「新建」
   - 复制保存好你的 API Key（格式：`sk-xxxxxxxx`）

### 第二步：下载项目代码

```bash
# 把项目代码下载到你的电脑
cd /mnt/agents/output/app

# 或者用 git 克隆（如果你把代码推到了 git）
# git clone https://你的git仓库地址
```

### 第三步：配置环境变量

打开项目根目录下的 `.env` 文件，添加你的 Kimi API Key：

```bash
# 在 .env 文件末尾添加这一行
KIMI_API_KEY=sk-你的API密钥
```

> 注意：数据库连接已经自动配置好了（TiDB 云数据库，免费额度）

### 第四步：安装依赖并启动

```bash
# 进入项目目录
cd /mnt/agents/output/app

# 安装依赖（第一次需要）
npm install

# 推送数据库表结构（第一次需要）
npm run db:push

# 启动开发服务器
npm run dev
```

启动成功后，在浏览器打开：http://localhost:3000

## 方式二：让外网也能访问（公网部署）

### 方案 A：云服务器部署（推荐长期使用）

**推荐云服务商**：阿里云 / 腾讯云 / 华为云

**最低配置**：1核2G 即可，约 ¥50-100/月

```bash
# 1. 购买云服务器后，SSH 登录
ssh root@你的服务器IP

# 2. 安装 Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs

# 3. 安装 PM2（进程守护）
npm install -g pm2

# 4. 上传项目代码（用 scp 或 git）
# scp -r /本地项目路径 root@服务器IP:/opt/ai-chat

# 5. 进入项目安装依赖
cd /opt/ai-chat && npm install && npm run build

# 6. 设置环境变量
echo "KIMI_API_KEY=sk-你的API密钥" >> .env

# 7. 用 PM2 启动
pm2 start dist/boot.js --name "ai-chat"
pm2 startup
pm2 save

# 8. 配置 Nginx 反向代理（80端口→3000端口）
```

### 方案 B：Docker 部署（一键启动）

```dockerfile
# Dockerfile（放在项目根目录）
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["node", "dist/boot.js"]
```

```bash
# 构建并运行
docker build -t ai-chat .
docker run -d -p 3000:3000 --env KIMI_API_KEY=sk-你的密钥 ai-chat
```

### 方案 C：内网穿透（把本地服务暴露到公网）

如果你不想买服务器，可以用 **ngrok** 把本地服务暴露到公网：

```bash
# 安装 ngrok（macOS）
brew install ngrok

# 或者去 https://ngrok.com 下载

# 启动 ngrok（把本地的 3000 端口暴露到公网）
ngrok http 3000
```

ngrok 会给你一个公网地址，如 `https://abc123.ngrok.io`，
把这个地址嵌入到你的企业官网即可。

> 免费版 ngrok 每次重启地址会变，适合测试。
> 正式使用建议买服务器或 ngrok 付费版（固定域名约 $8/月）。

## 接入 Kimi Work 的说明

**Kimi Work** 是 Kimi 的桌面端产品，主要面向个人办公场景，目前没有提供直接的 API 接口。

本系统接入的是 **Kimi 开放平台 API**（`https://api.moonshot.cn/v1`），这是 Kimi 官方提供的企业级 API 服务，与 Kimi Work 底层使用的是同一个大模型。

如果你希望用 Kimi Work 来管理客服对话，目前可以通过以下方式间接实现：
1. 在 Kimi Work 中创建客服助手 Bot
2. 通过本系统的管理后台接管对话
3. 后续 Kimi 可能会开放 Work 的 API，届时可以直接对接

## 常见问题

**Q: 15 元用完后怎么办？**
A: 在 https://platform.moonshot.cn 充值，最低 ¥50。按量计费，用多少扣多少。

**Q: 没有 Kimi API Key 能用吗？**
A: 可以！没有配置 API Key 时，系统会只用知识库匹配，匹配不到就返回兜底回复。建议配置 Kimi API 获得更好的智能回复效果。

**Q: 数据库在哪里？**
A: 系统默认使用 TiDB Serverless 云数据库（免费额度），不需要你自己搭建数据库。

**Q: 我的电脑关机后还能用吗？**
A: 本地运行的话，电脑关机服务就停了。需要长期在线请用云服务器部署。

**Q: 可以接入其他 AI 吗？**
A: 可以！代码中 `callKimiAPI` 函数就是调用 AI 的地方，把 `api.moonshot.cn` 改成其他 AI 服务的地址即可（如 OpenAI、通义千问、文心一言等）。
