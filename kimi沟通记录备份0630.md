## 第三部分：人工接管功能（v5.0新增）

### 3.1 功能概述

v5.0 新增人工接管功能，允许管理员在必要时介入AI对话，直接与客户沟通。

**核心能力：**
- ✅ 访客可主动点击"转人工客服"按钮
- ✅ 管理员在后台实时接管对话
- ✅ AI自动暂停，人工回复实时推送到访客端
- ✅ 管理员可随时释放对话，恢复AI自动回复
- ✅ 飞书通知包含后台接管链接
- ✅ 访客列表显示人工模式状态（AI/人工/请求人工）

### 3.2 数据表设计

#### human_replies 表

```sql
CREATE TABLE human_replies (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  visitor_id text NOT NULL,
  content text NOT NULL,
  is_read boolean DEFAULT false,
  created_at timestamptz DEFAULT now()
);
```

#### 系统指令（特殊content值）

| content 值 | 含义 | 触发方 |
|------------|------|--------|
| `[SYSTEM:ADMIN_TAKEOVER]` | 管理员接管对话 | 后台管理员 |
| `[SYSTEM:ADMIN_RELEASE]` | 管理员释放对话 | 后台管理员 |
| `[SYSTEM:USER_REQUEST_HUMAN]` | 用户请求人工 | 访客点击按钮 |
| `[客户消息] xxx` | 人工模式下的客户消息 | 客服组件自动 |

### 3.3 工作流程

```
1. 访客点击"转人工客服"按钮
   → 插入 human_replies {visitor_id, '[SYSTEM:USER_REQUEST_HUMAN]', false}
   → 发送飞书通知提醒管理员
   → 访客看到"已为您转接人工客服，请稍等"

2. 管理员在后台"在线对话"页面看到请求
   → 访客卡片显示红色"求人工"标签
   → 点击访客 → 点击"接管对话"
   → 插入 human_replies {visitor_id, '[SYSTEM:ADMIN_TAKEOVER]', true}

3. 访客端轮询检测到接管指令（每3秒）
   → 切换为人工模式 isHumanMode = true
   → UI显示绿色"人工客服为您服务中"状态条
   → 显示"人工客服已接入，有什么可以帮您的？"

4. 访客发送消息
   → 保存到 chat_records（role='user'）
   → 同时保存到 human_replies {visitor_id, '[客户消息] xxx', false}
   → 由于 isHumanMode=true，不调用AI API

5. 管理员看到访客消息，发送人工回复
   → 保存到 human_replies {visitor_id, '回复内容', false}
   → 同时保存到 chat_records {visitor_id, role='ai', '[人工] 回复内容'}

6. 访客端轮询检测到人工回复
   → 在聊天窗口显示 "👤 回复内容"
   → 标记为已读

7. 管理员点击"释放对话"
   → 插入 human_replies {visitor_id, '[SYSTEM:ADMIN_RELEASE]', true}
   → 访客端轮询检测到，恢复AI模式
   → 显示"已转回AI客服，继续为您服务"
```

### 3.4 客服组件轮询逻辑

```javascript
// 每3秒检查一次是否有新的人工回复
function checkHumanReplies(){
  fetch(SUPABASE_URL+'/rest/v1/human_replies?visitor_id=eq.'+vid+'&is_read=eq.false&order=id.asc')
    .then(r=>r.json())
    .then(data=>{
      data.forEach(reply=>{
        if(reply.content==='[SYSTEM:ADMIN_TAKEOVER]'){
          isHumanMode=true;
          updateHumanUI(); // 显示绿色状态条
          return;
        }
        if(reply.content==='[SYSTEM:ADMIN_RELEASE]'){
          isHumanMode=false;
          updateHumanUI(); // 移除状态条
          return;
        }
        // 显示人工回复
        A('ai','👤 '+reply.content);
        markReplyRead(reply.id);
      });
    });
}
setInterval(checkHumanReplies, 3000);
```

```
   ↓
8. 访客端轮询检测到人工回复 → 显示在聊天窗口
   ↓
9. 管理员点击"🔓 释放对话"
   → 插入 human_replies {SYSTEM:ADMIN_RELEASE}
   ↓
10. 访客端恢复AI模式
    → 显示"已转回AI客服，继续为您服务"
```

```sql
CREATE TABLE chat_records (

);
```
